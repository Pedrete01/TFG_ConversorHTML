HTML Conversor

Esta aplicación es un conversor de imágenes a formato HTML y CSS. Mediante la ayuda de una IA, la imágen se transformará a formato HTML.

Con hacer un dibujo a mano alzada bastaría para que el sistema pueda crear una página web. La inteligencia artificial está preparada para que el usuario obtenga los resultados más parecidos a la imagen pasada.

¿Cómo funciona?

Esta aplicación permite elegir al usuario dos formas de covertir sus diseños a HTML:

Con la primera de las dos opciones, el usuario tendrá la posibilidad de importar una imagen desde su ordenador a la aplicación. Esta imagen será procesada, y en caso de detectar un diseño de una página web, la aplicación generará un código HTML con el diseño importado. Esta opción se encuentra disponible en la pestaña de "conversor" en el menú desplegable de la izquierda.

Con la segunda de las dos opciones, el usuario tendrá la posibilidad de activar su cámara para poder capturar los elementos de su alrededor, esta opción es útil si el usuario desea importar un diseño dibujandolo delante de la cámara. Esta opción se encuentra disponible en la pestaña de "Live" en el menú desplegable de la izquierda.
